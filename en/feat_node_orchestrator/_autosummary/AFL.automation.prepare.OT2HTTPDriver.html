make_align_script(filename: str)[source]#

Generate an Opentrons Python Protocol API script to verify alignment.

This script recreates the current deck state (modules, labware, instruments) and performs a movement to the top of well A1 for each labware using each pipette. This allows for visual verification of calibration and alignment.
